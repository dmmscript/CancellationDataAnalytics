<h1 align="center" style="font-weight: bold;">CancellationDataAnalytics</h1>
<p align="center">
<b>CancellationDataAnalytics</b> is a project dedicated to analyzing customer cancellation data to uncover key reasons behind churn and develop strategies for retention using Python.
</p>

<h2 style="font-weight: bold;">📍 Features</h2>

- <b>Data Analysis</b>: Explore and analyze custumer cancellation data.
- <b>Churn Prediction</b>: Identify patterns and predict future churn.
- <b>Retention Strategies</b>: Develoop actionable strategies to retain custumers.
- <b>Visualization</b>: Create insightful visualizations to communicate findings.

<h2 style="font-weight: bold;"> 💻 Technologies</h2>

- Python 3.11.9
- Pandas
- Plotly

<h2 style="font-weight: bold;">💡 How to Use</h2>

- Clone the repository:

```bash
git clone https://github.com/dmmscript/CancellationDataAnalytics.git
```

- Install dependencies:

```bash
pip install pandas
```

```bash
pip install plotly
```
